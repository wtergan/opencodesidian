# Daily Review

Conduct an end-of-day review to capture progress and set up tomorrow.

## Review Process

1. **Today's Activity**
   - Find all notes modified today
   - Identify new notes created
   - Review work across all projects

2. **Progress Assessment**
   - What was accomplished?
   - What got stuck or blocked?
   - What unexpected discoveries emerged?

3. **Capture Insights**
   - Key learnings from today
   - New connections discovered
   - Questions that arose

4. **Tomorrow's Setup**
   - Top 3 priorities
   - Open loops to close
   - Questions to explore

## Output

**Display-only synthesis** (no file created by default):
- Analyzes today's daily note (02_Areas/Journal/Daily/YYYY-MM-DD.md)
- Reviews inbox items from today
- Presents: accomplishments, insights, blocks, tomorrow's focus

Optional: Can create/update daily note file if requested.
